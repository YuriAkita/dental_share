# README
開発環境
---
・Ruby 3.0.1
・Rails 6.1.4

就業Tarmの技術
---
・devise
・コメント機能
・お気に入り機能

カリキュラム外の技術
---
・Active Storage
・Nokogiri

実行手順
---
```
$ git clone git@github.com:YuriAkita/dental_share.git
$ cd dental_share
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
